# 📚 Library Admin System

A web-based library management system built with Laravel, designed for administrators to manage books, users, and library operations efficiently. This system is made for the capstone project.

🌐 **Live Demo**: [https://library-admin.bps.edu.ph](https://library-admin.bps.edu.ph)

---

## 🚀 Features

- 📖 Manage Books (Add, Edit, Delete, Search)
- 👥 Manage Users and Roles (Admin, Staff, Students)
- 📊 Dashboard with Statistics and Reports
- 📅 Track Borrowing and Return History
- 🔐 Authentication and Role-based Authorization (using Spatie)
- 📥 Import/Export Books and Users (Excel/PDF)
- 📂 Upload and Manage Digital Resources

---

## 🛠️ Tech Stack

- **Backend**: Laravel 10+
- **Frontend**: Blade, Tailwind, JQuery
- **Database**: MySQL
- **PDF/Excel**: DOMPDF, PhpSpreadsheet
- **User Roles**: Spatie Laravel Permission
- **Authentication**: Laravel Breeze
